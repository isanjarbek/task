Реализовать Мини Интернет магазин
есть готовые API
 GET -- /api/products
    возвращяет массив из продуктов (вы можете от себя добавить еще дополнительные)
 GET -- /api/payment
    возвращяет массив доступных платежных систем с их конфигурациями
 POST -- /api/pay
    Basic Auth: 
        username: usr
        password: pwd
    Body Raw JSON:
        paymnet: ${выбранная платежная система}
        ...requiredFields (обязательные поля указанные в конфигурации платежной системы)
    в случаи ошибки возвращяет success: false и ошибку
    в случаи успеха возвращяет success: true
